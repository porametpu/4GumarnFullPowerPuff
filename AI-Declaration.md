## รายงานฉบับนี้ นำปัญญาประดิษฐ์มาใช้ในขั้นตอนต่อไปนี้
-   1)	ศึกษาคำสั่งใน robot นอกเหนือจากที่ศึกษา โดยมีการศึกษาการใช้ Test Teardown, Suite Teardown นอกเหนือจากการ setup library
-   2)	ศึกษาการกำหนด variables แบบที่ไม่กำหนดค่าชัดเจนแต่เป็นการกรอก input ใน terminal การใช้คำสั่งรัน
-   3)	ศึกษาการใช้ตัวอย่างคำสั่ง keywords แบบกำหนดการทำงานเป็นเซ็ตคำสั่ง และการกำหนดการใช้ log ขั้นตอนลง console เพื่อสะดวกต่อการอ่าน flow การทำงานรวมถึงการใช้ Arguments เก็บค่าที่ได้รับจาก keywords ที่เรียกใช้งาน
-   4)	ศึกษา library cron ที่ใช้กำหนดให้โปรแกรมทำการลบบัญชีผู้ใช้ตามช่วงเวลาที่ได้ตั้งไว้
-   5)	ใช้ในการตรวจเช็คแก้ไขปัญหา token ไม่ถูกส่งใน handleDelete() ซึ่งได้รับการตรวจเช็คและแก้ไขเรียบร้อยแล้ว
-   6)	ศึกษาการจัดการข้อผิดพลาด (Error Handling) การกำหนด Response ตามมาตรฐาน
-   7)	ช่วยสร้างไฟล์ทดสอบด้วย Robot Framework จากกรณีทดสอบที่ได้ออกแบบไว้
