# CHANGELOG

## [2026-03-04] การปรับปรุงฟีเจอร์

### [ระบบส่งออกประวัติการสนทนา]
*Product Backlog Item No.18: As an admin, I want to export a chat log of particular users to support investigation when there is an official request from an authorized personnel.*

#### การปรับปรุงด้านรายงานและการส่งออกข้อมูล
- เพิ่มฟีเจอร์สำหรับ Admin ในการส่งออกข้อมูล Chat Log ของผู้ใช้ที่ระบุ
- เพิ่มการเลือกผู้ใช้ (Single / Multiple Users) สำหรับการดึงข้อมูล
- เพิ่มตัวกรองช่วงเวลา (Start Date – End Date) สำหรับจำกัดขอบเขตข้อมูล
- เพิ่มฟิลด์บังคับกรอก Official Request Reference ก่อนดำเนินการส่งออก
- พัฒนาฟังก์ชันการสร้างไฟล์รายงานในรูปแบบ PDF

#### การควบคุมสิทธิ์และความปลอดภัย
- จำกัดสิทธิ์การเข้าถึงฟังก์ชันส่งออกเฉพาะผู้ใช้งานที่มีบทบาท Admin
- เพิ่มระบบบันทึก Audit Log สำหรับการส่งออกข้อมูล
- ป้องกันการส่งออกข้อมูลหากไม่ได้กรอก Official Request Reference

#### การปรับปรุงโครงสร้างฐานข้อมูล
- เพิ่มตาราง audit_logs สำหรับบันทึกประวัติการส่งออกข้อมูล
- ปรับปรุงโครงสร้าง chat_messages เพื่อรองรับการค้นหาด้วยช่วงเวลา
- เพิ่ม index สำหรับเพิ่มประสิทธิภาพการดึงข้อมูล

#### การปรับปรุง UI/UX
- เพิ่มหน้า Export Chat Log สำหรับ Admin
- เพิ่มช่องกรอก Official Request Reference พร้อม validation
- ปรับปรุงข้อความแจ้งเตือนเมื่อข้อมูลไม่ครบถ้วน
- ปรับปรุงการแสดงผลรายงานก่อนดาวน์โหลดง
