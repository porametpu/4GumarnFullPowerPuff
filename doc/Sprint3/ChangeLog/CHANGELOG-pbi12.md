# CHANGELOG

## [2026-03-16] การปรับปรุงฟีเจอร์

### [ระบบแจ้งเตือนผู้โดยสารก่อนถึงจุดรับ]
*Product Backlog Item No.12: As a passenger, I want to get a notification when the driver is about to pick me up so that I can get myself ready or respond to the driver.*

#### การปรับปรุงด้านการทำงานหลัก
- เพิ่มเงื่อนไขให้ผู้โดยสารเลือกจุดรับภายในระยะ 5 กิโลเมตร เพื่อให้สามารถใช้งานปุ่มแจ้งเตือนได้  
- เพิ่มการตรวจสอบตำแหน่งล่าสุดของผู้ขับขี่ผ่านปุ่ม Refresh ในหน้า Hub  
- ปรับสถานะปุ่มแจ้งเตือนให้เปลี่ยนเป็นสีเหลืองเมื่อผู้โดยสารอยู่ในระยะที่กำหนด  
- เพิ่มฟังก์ชันให้ผู้ขับขี่สามารถส่งการแจ้งเตือนไปยังผู้โดยสารก่อนถึงจุดรับ  

#### การปรับปรุงด้านการแจ้งเตือน
- เพิ่ม Popup แจ้งเตือนบนหน้าจอของผู้โดยสารเมื่อได้รับการแจ้งเตือน  
- ปรับปรุงการแสดงผลสถานะการแจ้งเตือนให้ตอบสนองทันทีหลังจากกดส่ง  

#### การปรับปรุงเมนูและการเข้าถึงข้อมูล
- รองรับการตรวจสอบคำขอจองผ่านเมนู **คำขอจองเส้นทางของฉัน**  
- เพิ่มหน้ารายละเอียดการจองเพื่อแสดงข้อมูลผู้โดยสารในแต่ละรายการ  
- ปรับปรุงการแสดงผลสถานะคำขอจองแบบ Real-time  

#### การปรับปรุงการทดสอบระบบ
- ทดสอบการสร้างเส้นทางจากบัญชี Driver  
- ทดสอบการจองเส้นทางจากบัญชี Passenger  
- ทดสอบการรับคำขอจองและการกดส่งแจ้งเตือนจากฝั่ง Driver  
- ตรวจสอบการแสดงผลแจ้งเตือนหลังจากส่งสำเร็จ
